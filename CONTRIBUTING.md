# Contributing

Thanks for contributing to Triale

## Setup

git clone https://devmcsam/Triale
cd Triale
cargo build

## Style

Please run cargo fmt before contributing.
This project does not require dependencies so please try to keep 0 deps.
Please follow the general aim of the project to over engineer simpl stuff.

## Test

Please add tests for code that has possibilities of edge cases. An example of code that requires tests is the geometry math code.
An example of code that does not require tests is the string formatting that is dont for printing the TriangleSummary struct for example.
